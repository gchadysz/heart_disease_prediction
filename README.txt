Heart disease presence or lack thereof detection project. Using several machine learning classification models.


The dataset description can be found on:
https://archive.ics.uci.edu/ml/datasets/Heart+Disease

The dataset can be downloaded from:
https://archive.ics.uci.edu/ml/machine-learning-databases/heart-disease/

The file on which this project was based is named "processed.cleveland.data".

The archive contains both html version of the project as well as jupyter notebook file.
Both files are identical content-wise and contain description of the data, calculations,
but also analysis of results and the thought process along the way.

How to use notebook file:
1) Download the file "processed.cleveland.data".
2) Put it in to your current working directory.
3) Run the grzegorz_chadysz.ipynb file.

Note that some of the later calculations may take some time.
